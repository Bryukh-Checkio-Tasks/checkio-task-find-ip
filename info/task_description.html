You are given a text with IP-addresses. You should find all correct IPv4 addresses (Read about it). We don't know about subnet mask, so addresses ended by 0 and 255 are correct for us. All "words" in the text separated by whitespaces.

Your function should return a list of strings in the order how these IPs appears in the text.

Input: A text. A string (unicode).

Output: IPs. A list of strings.

Example: 

checkio("192.168.1.1 and 10.0.0.1 or 127.0.0.1") == ["192.168.1.1", "10.0.0.1", "127.0.0.1"]
checkio(u"10.0.0.1.1 but 127.0.0.256 1.1.1.1") == ["1.1.1.1"]
checkio("167.11.0.0 1.2.3.255 192,168,1,1") == ["167.11.0.0", "1.2.3.255"]
checkio(""00250.00001.0000002.1 251.1.2.1") == ["251.1.2.1]

How it is used: Text or configuration files parsing.
